# Manual de Usuario - Earth Vibe

## 1. Objetivo del Sistema
**Earth Vibe** es una plataforma integral diseñada para fomentar la sostenibilidad y el reciclaje dentro de la comunidad. Su objetivo principal es gamificar el proceso de reciclaje, permitiendo a los usuarios escanear productos, completar desafíos ecológicos y ganar recompensas canjeables, creando así un hábito positivo hacia el cuidado del medio ambiente.

## 2. Perfil del Usuario
El sistema está diseñado para ser intuitivo y accesible para:
*   **Usuarios Generales (Estudiantes/Comunidad):** Personas interesadas en reciclar, participar en retos y obtener beneficios. No requieren conocimientos técnicos.
*   **Administradores:** Encargados de gestionar el inventario de recompensas, validar desafíos y supervisar la actividad de la plataforma.

## 3. Requisitos del Entorno
Para utilizar Earth Vibe, necesitas:
*   **Dispositivo:**
    *   Un Smartphone (Android/iOS) para la App Móvil.
    *   O una Computadora/Tablet con navegador web moderno (Chrome, Firefox, Edge) para la versión Web.
*   **Conexión:** Acceso a Internet estable (Wi-Fi o Datos Móviles).
*   **Cámara:** Funcional para el escaneo de códigos de barras de productos.

## 4. Pasos para Usar el Sistema

### 4.1. Acceso y Registro
1.  **Ingresa a la plataforma:** Abre la aplicación móvil o navega a la dirección web proporcionada.
2.  **Crea tu cuenta:**
    *   Selecciona "Registrarse".
    *   Ingresa tus datos básicos (Nombre, Correo, Contraseña).
    *   Confirma tu cuenta si es necesario.
3.  **Inicia Sesión:** Usa tus credenciales para acceder al panel principal.

### 4.2. Pantallas y Funciones Principales

#### A. Inicio (Dashboard)
*   **Resumen:** Visualiza tus puntos actuales ("Eco-Points") y tu nivel de usuario.
*   **Notificaciones:** Recibe alertas sobre nuevos desafíos o recompensas disponibles.

#### B. Escáner de Productos (Reciclaje)
*   **Función:** Identifica productos reciclables mediante su código de barras.
*   **Uso:**
    1.  Ve a la sección de **Escáner**.
    2.  Apunta la cámara al código de barras del producto (botella, envase, etc.).
    3.  El sistema consultará **OpenFoodFacts** para validar el producto.
    4.  Si es válido, confirma el reciclaje para ganar puntos.

#### C. Desafíos (Challenges)
*   **Función:** Lista de retos ecológicos activos (ej. "Recicla 5 botellas hoy").
*   **Uso:**
    1.  Selecciona un desafío de la lista.
    2.  Sigue las instrucciones para completarlo.
    3.  Al finalizar, los puntos se sumarán automáticamente a tu perfil.

#### D. Tienda de Recompensas (NaziShop/Vibe Shop)
*   **Función:** Canjea tus puntos acumulados por premios reales o virtuales.
*   **Uso:**
    1.  Navega a la tienda.
    2.  Explora el catálogo de productos disponibles.
    3.  Selecciona "Canjear" en el artículo deseado.
    4.  Recibirás un código o confirmación para reclamar tu premio.

### 4.3. Flujo de Uso Completo (Ejemplo)
1.  **Inicio:** Abres la app y ves que tienes 50 puntos.
2.  **Acción:** Tienes una botella de agua vacía. Vas a la pestaña "Escáner".
3.  **Proceso:** Escaneas el código de barras. La app reconoce "Botella de Agua 500ml".
4.  **Recompensa:** La app te otorga +10 puntos por reciclar. Ahora tienes 60 puntos.
5.  **Canje:** Vas a la tienda y ves que un cupón de descuento cuesta 60 puntos. Lo canjeas.
6.  **Fin:** Recibes tu cupón y tu saldo se actualiza.
